The Box Model

What is the box model?

The box model is a box consisting of 3 layers that wraps around the content of every HTML element. The three layers are (starting at the inner-most layer): padding, border, and margin.

What is the content?

Content is the inner-most part of an element, it is (usually) the smallest possible box that that can be wrapped around the content written in by the developer. For text elements, this is the smallest possible box you can wrap around whatever font is being used (Note - a lot of fonts give their letters hard coded spacing, so it's not always to the exact edge of each letter). For images, this is usually just the smallest box that fits around the image.

By default, the width and height attributes control JUST this box - meaning that padding, border, and margin sizes are added on to the specified width or height.

What is the padding?

Padding is what separates the content from its borders. It serves to make an element larger without having to actually modify the size of its content. Most of the time the padding will be the same background color as the content, which is why you cannot set a specific background-color for padding - it is an extension of the element.

What is the border?

A border is what denotes the edge of an element, and separates the element from the margin. Unlike padding, a border can be styled in a many ways; border style (solid, dashed, dotted, etc), border color, and border width are all properties which can be modified.

What is the margin?

The margin is the outermost part of the box model, it serves to give an element personal space from the elements around it. Visually it is separated from the element and helps to make up the bulk of the "white space" of most documents, but know that all margins are wrapped around an element somewhere.
