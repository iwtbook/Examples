Inline vs. Block Elements

Block Elements

These are block level elements

You'll notice that they default to filling up the entire width of their container

But even when they don't fill in the whole width...

They still create a new line

Inline Elements

This paragraph has a lot of inline elements. As you can see, these elements don't break the flow of the paragraph.

The size of inline elements is instead defined by their contents. Unlike block elements, you cannot set the width, height, margin-top, or margin-bottom of inline elements.

Inline-Block Elements?

Inline block elements are very similar to regular inline elements, except they respect
width, height, margin-top, and margin-bottom.

There are at least a dozen other display types as well that we'll get to in other demos.
