Inline vs. Block Elements

About element display types

What is a block level element?

Block level elements have two main characteristics: They (by default) will fill up the entire width of their container, and they will insert a return, causing the following element to be displayed on the next line.

What is an inline level element?

Inline level elements are almost the opposite of block level elements. They take up just as much width as their contents need, and they do not cause anything to return to a new line.

Additionally, they ignore four CSS rules: width, height, margin-top, and margin-bottom (even if included in a shorthand margin rule).

There's inline-block??

Inline-block level elements are the same as inline elements, except they notably allow you to modify the width, height, and all four margins.

There are at least a dozen other display types as well that we'll get to in other demos.
