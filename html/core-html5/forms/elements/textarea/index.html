The <textarea> Element

With just regular <input> elements, you are limited to just one line, no matter how you style them.

To receive multiline user input, you need to use the <textarea> element.

Basic textarea:

Textareas also have some unique features:

  • They are whitespace sensitive of user input
  • For added security, they do not render any HTML inside them
