Bidirectional Content

<bdi> - bidirectional isolate

This element allows you to isolate the text within from the bidirectional algorithm when it parses the surrounding text. Useful for situations where you aren't sure about the directionality of the content you are inserting.

This text is forewards. גוּט טַק אִים בְּטַגְֿא שְ וַיר דִּיש מַחֲזוֹר אִין בֵּיתֿ הַכְּנֶסֶתֿ טְרַגְֿא. This text is forewards again.

<bdo> - bidirectional text override

This element allows you to override the standard directionality of the content you are adding to the page.

This text is forewards. This text is backwards.
