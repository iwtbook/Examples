Urgent & Emphasized Content

The <b> tag

This text is bold for emphasis. This <b> tag has no semantic use outside of drawing the reader's attention to its contents. It should not be used to style text.

The <i> tag

This text is italicized and represents text that is set off from the normal text for some reason. This <i> tag has no semantic use.

The <strong> tag

This text is also bold. It represents content that is important or urgent, meaning that this <strong> tag has semantic use (unlike its <b> counterpart).

The <em> tag

This text is also italicized. It represents content that is to be stressed or emphasized, meaning that this <em> tag has semantic use (unlike its <i> counterpart).
