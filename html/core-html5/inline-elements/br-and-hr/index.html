Break Elements

<br />

<br /> is an empty element, which is why it's written often as a self-closing tag using XHTML synax.

Previously, a lot of people used <br /> elements to create spacing between elements or new paragraphs within a paragraph tag

like this. These use cases have fallen out of favor for more semantic uses, such as creating new lines in poetry or formatting an address.

<hr />

<hr /> is an empty element as well, so it is written with the self closing style.

The <hr /> element formerly stood for "Horizontal Rule", but in an effort to give it a more semantic meaning and less visual, it has since been renamed to the "Thematic Break" element.

As you can see above, it usually renders as a thin horizontal line by default.
