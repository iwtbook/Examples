Data URIs

What are Data URIs?

Data URIs are a way of embedding small amounts of content / files entirely into a single URI. The URI does not point to any server like a traditional URI, but instead contains all of the content within itself, usually base64 encoded to save space.

Data URIs have 3 parts, with an optional 4th part. In order they are:

data:[<mediatype>][;base64],<data>
  1. a data: prefix in the protocol spot
  2. the MIME type of the data
  3. (optional) a base64 token as a flag if the content is encoded
  4. the data itself.

They are also sometimes referred to as Data URLs.
