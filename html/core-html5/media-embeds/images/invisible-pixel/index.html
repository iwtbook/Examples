Invisible Pixel Layout

What's the point of a 1-pixel gif?

Before CSS was wide spread and had as many features as it does today, many developers had to use 1-pixel gifs to push text and divs around to properly lay them out. It's a technique you might see on older websites today.

The 1-pixel gif was chosen as you needed a block-level <img> tag to be able to size and push around content, but you also needed the server to respond with something so the tag would render properly. The gif format supported transparency and was one of the smallest file sizes to use for this purpose.
