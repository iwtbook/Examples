Comments

HTML Comments usually take the form as such:

<!-- This is an HTML I escaped so it could render on screen -->

Of course, the front end also deals with two other technologies that allow for comments. Below is an example of a CSS comment:

/* This is a CSS comment */

And finally JavaScript, which allows for two types of comments
